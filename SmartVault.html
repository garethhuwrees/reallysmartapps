Smart Vault

A simple to use password store

If you are one of these people who store there password in the back of their diary - please dont.

Key Features

  • Log-in with a single 4 digit passcode (PIN).
  • Log and view all your passwords in a single list.
  • Copy passwords to clipboard to paste into website
  • Set default user name (usually your email), which can be overwritten if necassary
  • Classic and jailhouse background themes
  • Supports iPhone Light and Dark modes.

Introduction

Smart Vault is deliberately designed to be simple with just the minimal fuctionality required to store your internet passwors on your iPhone. It is intended to encourage those people that carry around written passwords to raise their on-line security up one notch.

If you required rich functionality to manage your on-line access then there are a number of excellent apps in the App Store that will do this for you.
