Smart Pass

An easy to use passcode manager - with a twist!

Designed to help improve password hygiene but but with the minimum administative overhead as possible.

Designed to help those people who have 'traditionally' used the same password for multiple sites adopt more disciplined approach to on-line security. This is achieved by having the minimum administative overhead as possible.

Overview

Are you someone who reuses the same password across multiple sites? I was, until one of these was hacked and 'strange' things started happening on other sites.

I initially turned to one of the many excellent comprehensive password management applications on the App Store but unfortunately I wasn't disciplined enough to maintain the data. So, I created SmartPass.

The App doesn't hold any passwords, but uses 2 bits of fixed data (a PIN and 'memorable' word) plus one variable (the domain or application name) to generate a 'sudo-random' passcode. When you need it again, it simply regenerates it.

Go on - give it a go! The App is free to use although if you find it useful you may want to show your appreciation by buying me a coffee.

Key Features

  • Log-in with a single 4 digit PIN.
  • Enter the domain or application name to generate the initial passcode
  • The passcode and user name is stored in a table
  • A simple touch of the table entry retrieves the passcodey
  • Passcodes are akways copied to the 'pasteboard'
  • Backup entries to iCloud
  • Classic and jailhouse background themes
  • Supports iPhone Light and Dark modes.

Instructions

Set Up

Set the ‘Key Data’ - a 4 digit PIN and a 8 (or more) character memorable word or catch phrase. These, along with the domain name, are used to calculate the passcode. Also (optionally) set a default username.

Add Passcodes

Set requirements for passcode length and the use of optional characters. Then enter the domain name and username (unless using the default) and touch ‘Generate Passcode’. These settings are saved under the Passcodes tab.

Passcodes

Touch a domain name to reveal the passcode. Use a long press to change the username. Swipe left to delete a record.

General

Remember to periodically to backup the passcode - under settings. This does require an iCloud account.
